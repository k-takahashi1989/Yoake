## 実装メモ

### 推奨実装方法（react-native-svg + ローカル SVG）

```bash
# パッケージは既存の react-native-svg を使用
# Flaticon からSVGをダウンロード → src/assets/icons/ に配置
```

```tsx
// src/components/common/Icon.tsx (新規作成)
import MoonIcon from '../../assets/icons/moon.svg';
import BookIcon from '../../assets/icons/book.svg';
// ... etc.

export type IconName = 'moon' | 'book' | 'chart' | ...;

const ICONS: Record<IconName, React.FC<SvgProps>> = {
  moon: MoonIcon,
  book: BookIcon,
  // ...
};

export function Icon({ name, size = 24, color = '#FFFFFF' }: Props) {
  const Component = ICONS[name];
  return <Component width={size} height={size} fill={color} />;
}
```

### タブバーへの適用

```tsx
// navigation/index.tsx
function TabIcon({ name, focused }: { name: string; focused: boolean }) {
  const iconMap: Record<string, IconName> = {
    Home: 'moon', Diary: 'book', Report: 'chart', Alarm: 'alarm', Profile: 'user',
  };
  return (
    <Icon
      name={iconMap[name]}
      size={22}
      color={focused ? '#6B5CE7' : '#9E9E9E'}
    />
  );
}
```

### 注意点
- Flaticon の outline SVG は `fill` 属性で色を制御できるものと `stroke` で制御するものが混在するため、ダウンロード時に確認すること
- アイコンスタイルは **全て同一スタイルセット** から選ぶこと（line / flat / gradient を混在させない）
- 推奨スタイル: **"Lineal" または "Outline"**（Flaticon でフィルタ可能）
- サイズは 24x24px でエクスポートし、アプリ側でリサイズ

### 選定基準

- **ループなし**: 達成感・演出系（チェック・sunrise）
- **ループあり**: ローディング・空状態系（AI生成中・空日記）
- **サイズ**: 120px 以下のシンプルなもの（ファイルサイズ 50KB 以内推奨）
- **スタイル**: 白・紫・青系の配色（背景 `#1A1A2E` に映える）
- **ライセンス**: Free / Lottie Simple License（商用利用可）で確認
